## Symlinking the files


```bash
ln -s ~/dotfiles/.zshrc  ~/.zshrc
```


## Needs to be linked:

```bash
ln -s ~/dotfiles/nvim  ~/.config/nvim
```


```bash
ln -s ~/dotfiles/.ipython  ~/.ipython
```

## Notes

If git is too old, lazyvim will not install correctly. 
You can use conda to install a new git.
