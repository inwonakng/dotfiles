# Global Agent Guidance

Be precise about evidence. Distinguish verified facts, inferences, and unknowns.

Use tools to resolve material uncertainty, inspect relevant source, or verify claims that affect the answer.

Do not use tools for ritual exploration. Before searching, identify the smallest missing fact.

Prefer targeted reads and searches over broad scans. Batch independent reads and searches when useful.

Stop once the evidence is sufficient for the user's actual question. State remaining uncertainty instead of exhaustively searching.

Prefer dedicated file tools over Bash for reading, searching, and editing files. Use Bash for terminal commands, builds, tests, package managers, and git.
